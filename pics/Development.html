PROJECT 3 – DEVELOPING BELCO ALLIANCE WEBSITE

Creative Animation, User Testing & Collaborative Development

After completing the prototyping and analysis phase of the Belco Alliance website redesign, our class was divided into smaller teams of three. I worked in a group with one other student, and together we focused on developing a creative and functional new version of the Belco Alliance website. We wanted to make a strong visual impact right from the beginning, so we decided to use an engaging animation concept for the cover page—an interactive “open the door” effect. This idea was chosen to reflect the welcoming and inclusive values of the Belco Alliance.

1. Team Charter

I was responsible for creating our team charter, which outlined clear group norms, expectations, responsibilities, communication methods, and deadlines. The charter played a crucial role in helping us stay organized and work efficiently. It ensured that both of us remained accountable and that our workflow was structured and collaborative. By having a mutual understanding of our roles and the timeline, we were able to avoid delays and complete each task on schedule.

2. Git Upload

We used Git to manage our code collaboratively, which was essential for version control and tracking progress. However, we encountered some technical issues while uploading our files to GitHub, such as merge conflicts and syncing errors. Despite these challenges, we resolved the problems by seeking support and exploring Git documentation, which helped us gain a deeper understanding of how Git works.

3. 3D Door Animation

One of my main contributions to the project was designing and coding a 3D door animation effect to be used as the opening cover page of the Belco Alliance website. The concept was to create a symbolic gesture of “entering” a safe and supportive space, aligning with Belco’s mission. I used HTML, CSS, and JavaScript to create this immersive animation, which adds an engaging and interactive element to the user’s first impression of the site.

Link to LO1: View Door Animation - Check Proof 3

4. Survey Design

To ensure our animation and design ideas resonated with users, I created a detailed online survey to gather feedback from potential users and stakeholders. The survey focused on visual elements, animation preferences, usability, and overall impressions of the redesigned site concept. The responses we collected provided valuable insights that influenced our design decisions and improvements.

Link to LO4: View Survey - Check Proof 5

5. User Testing Questions

Before we started the actual coding phase, we conducted user testing to evaluate how effective and appealing our animation concept would be. I developed a set of user testing questions specifically related to the animation, including questions about clarity, speed, aesthetics, and emotional impact. The feedback helped us determine whether the animation was intuitive and engaging, and what changes were necessary to improve the user experience. This early testing phase was crucial in helping us align our design with user expectations.
